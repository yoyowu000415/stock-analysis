【准备工作】（只需做一次）

  1. 安装 Python 3.11+
     下载: https://www.python.org/downloads/
     ★ 安装时务必勾选 "Add Python to PATH"

  2. 安装 Node.js（用于 Web 界面）
     下载: https://nodejs.org/
     选择 LTS 版本即可

  3. 获取 DeepSeek API Key（AI 分析引擎）
     注册: https://platform.deepseek.com
     注册后进入 "API Keys" 页面 → 创建 API Key
     充值几块钱即可使用很久

【启动方式】

  双击运行「一键启动.bat」

  首次运行会自动：
    → 安装 uv（Python 包管理器）
    → 创建虚拟环境
    → 安装所有依赖（约 136 个包，需几分钟）
    → 提示你输入 DeepSeek API Key
    → 启动 Web 界面

  启动成功后浏览器打开: http://127.0.0.1:8000

【功能说明】

  ● Web 界面可以：
    - 添加/管理自选股（A股、港股、美股）
    - 手动触发 AI 分析
    - 查看历史分析报告
    - 修改系统配置

  ● 支持的股票代码格式：
    - A股沪市: 600519（贵州茅台）
    - A股深市: 000001（平安银行）
    - 创业板:  300750（宁德时代）
    - 港股:    00700（腾讯）
    - 美股:    AAPL（苹果）

【常见问题】

  Q: 提示"未检测到 Python"？
  A: 安装 Python 时一定要勾选 "Add Python to PATH"，安装后重启电脑

  Q: 依赖安装失败？
  A: 检查网络连接，必要时开启科学上网

  Q: 分析结果为空？
  A: 检查 DeepSeek API Key 是否正确，账户是否有余额

  Q: 想修改自选股？
  A: 在 Web 界面的设置页面修改，或编辑 .env 文件中的 STOCK_LIST

【停止服务】

  在命令行窗口按 Ctrl+C
